## 🚀 Installation

```bash
pip install weather-app-mannyyebz
```

## 🔑 How to Integrate API Key

```bash
cp .env
# Add your OpenWeatherMap API key to .env
WEATHER_API_KEY=your_key_here
```

## ▶️ How to Run

```bash
weather-app Tokyo
```
